1. Create a GitHub Repository

Go to github.com

Click New Repository

Give your repo a name

Choose Public or Private

Hit Create Repository

2. Initialize Git in Your Project Folder

Open your terminal inside your project folder:

git init
git add .
git commit -m "Initial commit"

3. Connect Local Project to GitHub

Copy the repo URL from GitHub.

git remote add origin https://github.com/username/repo-name.git


Check connection:

git remote -v

 4. Push Your Project to GitHub

If you're on main branch:

git branch -M main
git push -u origin main


If you get an error like rejected – fetch first:

git pull origin main --allow-unrelated-histories
git push origin main

OPTIONAL: Deploy Website Using GitHub Pages

(For HTML/CSS/JS static websites)

5. Enable GitHub Pages

Open your repo

Go to Settings → Pages

Under Build and Deployment, choose:

Source: Deploy from Branch

Branch: main → /root

Save

Wait 30–60 seconds.
Your site will be live at:

https://username.github.io/repo-name/
