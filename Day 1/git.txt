Complete Git Commands List
 1. Setup & Configuration
Command	Description
git --version	Check installed Git version
git config --global user.name "Your Name"	Set username
git config --global user.email "your@email.com"	Set email
git config --list	Show all Git config values
git help	Show help menu

 2. Create / Initialize Repository
Command	Description
git init	Initialize a new Git repository
git clone <url>	Clone/download a repository

 3. Staging & Committing
Command	Description
git add <file>	Add a file to staging area
git add .	Add all files to staging area
git commit -m "message"	Commit with message
git commit -am "message"	Add & commit tracked files
git status	Check status of files
git diff	Show changes before commit

4. Branching
Command	Description
git branch	List all branches
git branch <branch_name>	Create new branch
git branch -d <branch_name>	Delete branch
git checkout <branch>	Switch branch
git checkout -b <branch>	Create & switch to branch
git switch <branch>	Switch using new command
git switch -c <branch>	Create & switch using new command

5. Merging & Rebasing
Command	Description
git merge <branch>	Merge a branch into current
git rebase <branch>	Reapply commits over branch
git merge --abort	Abort a merge conflict
git rebase --abort	Abort rebase

 6. Remote Repository Commands
Command	Description
git remote -v	View remote URLs
git remote add origin <url>	Add remote repo
git remote remove origin	Remove remote
git push origin main	Push to main branch
git push -u origin main	Set upstream branch
git pull	Pull latest changes
git fetch	Download updates (no merge)

 7. Viewing History
Command	Description
git log	Show commit history
git log --oneline	Short commit history
git show <commit_id>	Show details of a commit

 8. Undo / Restore / Fix
Command	Description
git restore <file>	Restore file to last commit
git restore --staged <file>	Unstage a file
git reset <file>	Remove from staging
git reset --hard	Undo all changes permanently
git checkout <commit>	Move to older version
git revert <commit>	Undo commit safely by creating new commit

 9. Stashing
Command	Description
git stash	Save uncommitted changes temporarily
git stash list	Show all stashes
git stash apply	Apply stash
git stash drop	Delete stash

 10. Tagging
Command	Description
git tag	List tags
git tag <tag>	Create tag
git tag -a v1.0 -m "version 1.0"	Create annotated tag
git push origin --tags	Push tags to GitHub

 11. Remove Files
Command	Description
git rm <file>	Remove file from repo
git rm -r <folder>	Remove folder

 12. Advanced Commands
Command	Description
git cherry-pick <commit>	Apply a commit from another branch
git blame <file>	Show who last modified each line
git bisect	Find buggy commit using binary search
